In this assignment I practiced recursion in Java and solved different tasks without using loops. In task 1 printed each digit of a number using recursion, task 2 found sum and average of array elements,task 3 checked if a number is prime, task 4 calculated factorial, task 5 found Fibonacci number, task 6 calculated power of a number, task 7 Printed numbers in reverse order,task 8 Checked if string contains only digits,Task 9 Counted number of characters in string,Task 10 Found GCD using recursion.This assignment helped me better understand how recursion works and how to use it in different problems.
